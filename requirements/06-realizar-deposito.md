# Realizar deposito.

Como cliente, gostaria de realizar um deposito bancario.

> ## Caso de sucesso

1. ✅ Recebe uma requisição do tipo **Post** na rota **/api/**
2. ✅ Valida se o usuário de origem é valido
3. ✅ Retorna **200**.

> ## Exceções

1. ✅ Retorna o erro **400** caso o usuário não seja valido!
